# Análisis de Datos y Ajuste de Modelo Final

## Descripción

Este repositorio contiene el desarrollo del análisis de datos realizado con el conjunto de datos asignado. Incluye la preparación y organización de los datos, análisis exploratorio, aplicación del modelo seleccionado, evaluación de resultados y presentación de tablas y gráficos obtenidos durante el proceso.

## Estructura del proyecto

**Datos:**  
Contiene la base de datos utilizada para realizar el análisis.

**Código:**  
Incluye los archivos con los procedimientos y comandos empleados para la limpieza, procesamiento, análisis y ajuste del modelo.

**Resultados:**  
Contiene las tablas, gráficos y resultados obtenidos a partir del modelo final aplicado.

**Informe final:**  
Documento donde se describe el proceso desarrollado, la metodología utilizada y la interpretación de los resultados obtenidos.
