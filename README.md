Análisis de Datos y Ajuste de Modelo FinaL
Descripción
Este repositorio contiene el desarrollo del análisis de datos realizado con el conjunto de datos asignado. Incluye la preparación de datos, análisis exploratorio, aplicación del modelo seleccionado, evaluación de resultados y presentación de tablas y gráficos obtenidos.
Estructura del proyecto
Datos: Contiene la base de datos utilizada para realizar el análisis.
Código: Incluye los archivos con los procedimientos y comandos utilizados para la limpieza, análisis y ajuste del modelo.
Resultados: Contiene las tablas, gráficos y resultados obtenidos del modelo final.
Informe final: Documento donde se explica todo el proceso desarrollado y la interpretación de los resultados.
Herramientas utilizadas
Programa estadístico utilizado: R / Python
Plataforma de almacenamiento y documentación: GitHub
