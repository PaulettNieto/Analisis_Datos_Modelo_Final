Problemática
En estudiantes de Psicología se observan niveles de estrés moderados y altos relacionados con las horas de estudio y las horas de sueño.
Objetivo general
Realizar un análisis descriptivo de la relación entre las horas de estudio, las horas de sueño y los niveles de estrés.
Proceso de análisis
1. Creación de la base de datos.
2. Identificación de datos perdidos.
Se introdujeron tres valores faltantes en la base de datos con fines de análisis. Los datos perdidos se ubicaron en las variables Horas_estudio y Horas_sueño.
Casos con datos faltantes:
ID 3: Horas_sueño.
ID 7: Horas_estudio.
ID 12: Horas_sueño.
4. Aplicación de imputación múltiple.
5. Comparación de estadísticas descriptivas.
6. Interpretación de resultados.
7. Conclusiones.
